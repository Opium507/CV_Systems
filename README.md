# CV_Systems
В данном репозитории собраны домашние и практические работы по дисциплине **«Системы технического зрения»**.
---
Цель репозитория — последовательно изучить и реализовать алгоритмы компьютерного зрения

## 📂 Оглавление и структура репозитория

Каждая домашняя работа находится в отдельной папке и имеет собственный `README.md` с подробным описанием логики алгоритма, инструкцией по запуску и примерами работы.

```text
CV_Systems/
│
├── HW_1/                  # ДЗ №1: Основы OpenCV и Tkinter
│   ├── test.py            # Базовый скрипт захвата видео с веб-камеры
|   ├── requirements.txt   # Зависимости для ДЗ №1
│   └── README.md          # Описание ДЗ №1
│
├── HW_2/                  # ДЗ №2: Программная замена экрана (AR)
│   ├── data/              # Исходные фото/видео и слайд презентации
│   ├── results/           # Результаты работы алгоритма
│   ├── HW_21.py           # Скрипт обработки статичных фото
│   ├── HW_22.py           # Скрипт обработки видео (со сглаживанием)
│   ├── requirements.txt   # Зависимости для ДЗ №2
│   └── README.md          # Описание ДЗ №2 (алгоритм, математика, примеры)
│
├── HW_N/                  # ДЗ №N: [Название будущей работы]
│   └── (будущее содержимое)
│
├── .gitignore             # Игнорируемые системой контроля версий файлы
└── README.md              # Текущий файл (общее оглавление)
```

## 🛠 Стек технологий
В рамках курса используются следующие инструменты и библиотеки:

- Python 3 (Основной язык разработки)
- OpenCV (Компьютерное зрение, обработка видеопотока, морфологические операции)
- NumPy (Матричные вычисления и работа с массивами данных)
- CustomTkinter (Создание графического интерфейса пользователя)

## 🚀 Как работать с репозиторием
1. Склонируйте репозиторий на свой компьютер:
   ```bash
   git clone https://github.com/Opium507/CV_Systems.git
2. Перейдите в папку нужной домашней работы (например, HW_2):
   ```bash
   cd CV_Systems/HW_2
3. Установите необходимые зависимости для конкретной работы:
   ```bash
   pip install -r requirements.txt
4. Изучите README.md внутри папки для получения инструкций по запуску скриптов.
