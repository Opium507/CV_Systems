## Домашняя работа №1 — Основы работы с OpenCV и Tkinter

### Что изучили
- Основы библиотеки OpenCV: чтение видео с камеры и из файла (`VideoCapture`), покадровая обработка, рисование примитивов (`rectangle`)
- Цветовые пространства и конвертация (`cvtColor`, BGR → RGBA)
- Обработка событий мыши через Tkinter `bind`
- Интеграция OpenCV с GUI-фреймворком CustomTkinter: вывод видеопотока в окне приложения, кнопки, обработка клавиатуры

### Задание (ТЗ)
Разработать приложение на Python (OpenCV + CustomTkinter), которое:
1. Принимает источник видео через аргумент командной строки:
   - `python test.py 0` — веб-камера
   - `python test.py video.avi` — видеофайл
   - При запуске без аргументов открывается диалог выбора источника
2. Отображает видеопоток в реальном времени в окне GUI
3. По нажатию ЛКМ на видео рисует зелёный прямоугольник в точке клика
4. По нажатию клавиши **C** или кнопки «Сбросить точки» — очищает все нарисованные прямоугольники
5. По нажатию клавиши **Q** или кнопки «Выход» — завершает приложение
